FAQ

  • addHow to start up?
    Well, it's quite simple. You need to install node in your machine. Just go the theme folder and command for NPM install. This will install all dependency for the theme. Now, you are ready to go for gulp it. However, I have mentioned other gulp configure commands that you can use in this theme here.
  • addWhat is the basic setup? Which file are mandatory?
    You can refer blank page from dist/pages/blank.html
  • addHow can I customize the theme or add new pages?
    It's easy. You can refer blank page template with sidebar menu. Add the option as per your requirements. For style customization, you can use custom.scss file.

Overview

Forge is admin template theme powered with Materialcss Framework (V 0.98.2). Forge can be used for any kind of application backend management. Forge has a sleek, clean and intuitive flat style design which makes your next project look awesome and yet user friendly. It has a huge collection of plugins and UI components and works seamlessly on all major web browsers, tablets and phones.

I have been putting lot of efforts to make it much useful tool for everyone with love.
