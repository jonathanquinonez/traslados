  });

Material Note

A Summernote wysiwyg editor version converted for Materialize.The only difference from Summernote is that you have to call materialnote() instad of summernote().

MaterialNote is a fork of Summernote wysiwyg editor (See the Original API) Edited by CK.


<div class="input-field col s12">
  <div id="first" class="editor"></div>
</div>
Demo

Toolbar

MaterialNote uses a javascript function to keep always visible the toolbar during scrolling. If you don't want this to be enabled, pass followingToolbar: false option to the constructor.

Paragraph buttons

The paragraph buttons are available in dropdown format like Summernote (['paragraph'] option in the toolbar array) or as separate buttons (['leftButton', 'centerButton', 'rightButton', 'justifyButton', 'outdentButton', 'indentButton']).


$(document).ready(function() {
  var toolbar = [
    ['style', ['style', 'bold', 'italic', 'underline', 'strikethrough', 'clear']],
    ['fonts', ['fontsize', 'fontname']],
    ['color', ['color']],
    ['undo', ['undo', 'redo', 'help']],
    ['ckMedia', ['ckImageUploader', 'ckVideoEmbeeder']],
    ['misc', ['link', 'picture', 'table', 'hr', 'codeview', 'fullscreen']],
    ['para', ['ul', 'ol', 'paragraph', 'leftButton', 'centerButton', 'rightButton', 'justifyButton', 'outdentButton', 'indentButton']],
    ['height', ['lineheight']],
  ];

  $('.editor').materialnote({
    toolbar: toolbar,
    height: 350,
    minHeight: 100,
    defaultBackColor: '#fff'
  });
});
