Overlay

Overlay is one the useful property. In general , we use it in many cases like - on a picture, loading event and much more. We have a huge collection of overlay using all material colors.

Defining Overlay

If you want an element has overlay then add .has-overlay, inside this element add another element .overlay. You can assign color and shaded to .overlay similary way *-overlay. Yeah, you also can set the shades overlay-lighten-* and overlay-darken-* or overlay-accent-*.
